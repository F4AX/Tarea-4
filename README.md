# Tarea-4: ALU Simple de 2 Bits

Este repositorio contiene el archivo de simulación de una ALU (Unidad Lógico Aritmética) de 2 bits, diseñada para la Tarea 4 de Hardware 
Digital. 

El circuito recibe dos números de 2 bits y, mediante dos bits de selección, permite ejecutar y mostrar el resultado de cuatro operaciones 
distintas en tiempo real: AND, OR, XOR y Suma con detección de acarreo.
